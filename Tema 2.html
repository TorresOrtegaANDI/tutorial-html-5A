Estilos de letras y diseño.

Para los diferentes estilos de letra tenemos algunas maneras de cambiar desde tamaño, hasta estilo. Esto te permite darle un gran disño unico a cada paguina web que deses creear. A continuacion te compartiremos algunos codigos que te serviran de mucha ayuda:

Para centrar tu texto deveras usar: < center >...< /center >, enmedio de este codigo deveras escribir el texto que quieras que este centrado.
EJEMPLO: Centrado

Para resaltar tu texto en negritas deveremos usar el codigo: < b >...< /b >, enmedio de este codigo escribiremos lo que queremos resaltar.

EJEMPLO: Texto en negritas

Para Poner un texto en Cursiva usaremos el codigo: < i >...< i >, este le dara una pequeña inclinacion a nuestro texto.

EJEMPLO: Texto en cursiva

Para poner el texto subrayado usaremos: < u >...< /u >. este le pondrea una pequeña linea, que dara el estilo de subrayado en nuestro texto.

EJEMPLO: Texto en subrayado

Para poner el texto tachado usaremos: < s >...< /s >, este tachara nuestro texto dandole un toque unico.

EJEMPLO: Texto en tachado

Para poner el texto enfatizado usaremos el codigo: < em >...< /em >.

EJEMPLO: Texto enfatizado

Para poner el Texto en forma de teletipo usaremos: < tt >...< /tt >, este le cambiara el estilo de letra.

EJEMPLO: Texto en forma de teletipo

Para poner el texto en grande usaremos: < big >...< /big >.

EJEMPLO: Texto en tamaño grande

Para poner el texto en pequeño usremos: < small >...< /small >.

EJEMPLO: Texto en tamaño pequeño

Para poner el texto en subindice ponemos: < sub>...< /sub>.

EJEMPLO: Subindice

Para poner el texto como superindice usaremos: < sup >...< /sup >.

EJEMPLO: Superindice

  6. < h6 >...< /h6 >, EJEMPLO:

Como viste estos codigos sirven para ajustar el tamaño de la letra esto te puede ayudar mucho, ya que es una detalle muy importante, el tamaño de los textos te puede ayudar a que sea mas entendible tu pagina web o para resaltar temas, subtemas, de todos lo demas que alla en tu texto.

Para terminar tambien quiero explicarte un tema muy importante que es el como podemos cambiar el diseño de nuestra pagina. Como hemos visto el lenguaje html es un lenguaje muy interesante, facil y muy practico para la programacion, es importante que sepas que para que tu puedas guardar una pagina como html, es importante que al momento de crear una nueva pagina la guardes como ".html", ya que si no haces esto no podras trabajar en este lenguaje una ves echo esto deveras iniciar tu programa con el codigo < html >, poniendo esto en tu pagina previamente guardada, te aparecera una opcion que dira html le precionaras y te dara el codigo.

Ya sabiendo esto podras crear una nueva hoja en la que esta vez deveras guardar como ".css" esto ara que esa pagina se guarde como un pequeño editor del diseño de tu pagina, hay podras empezar a trabajar en el diseño, pero deveras conectarla con el codigo, < link rel="stylesheet" type="text/css" href="nombre de tu pagina.css" >, este ira abajo del codigo < title >< /title >, una ves que pongas ese codigo en el apartado de (href="...css") deveras poner dentro de las comillas el nombre de la pagina a la cual le quieres dar el diseño, una vez haciendo eso solo sera que empiezes a diseñar y guardes los cambios, haci todo el codigo de diseño que ingreses en tu pagina ".css" se le aplicara a tu pagina guardada en ".html".
